# Journalistin — Portfolio

Diese kleine statische Website ist ein Portfolio‑Template für eine Journalistin mit editorialem Look: fette Sans‑Serif‑Überschriften (Inter), Fließtext in Courier New, asymmetrisches Grid und große Bilder.

Ordnerstruktur

journalistin/
├── index.html
├── style.css
├── script.js
└── images/
    ├── titelbild.svg
    ├── article1.svg
    ├── article2.svg
    ├── article3.svg
    └── portrait.svg

So geht's weiter

- Du kannst eigene Bilder in journalistin/images/ hochladen und die Platzhalter ersetzen.
- Wenn du möchtest, trage ich deinen Namen bzw. Kontaktinformationen ein.
- Ich kann die Startseite außerdem so erweitern, dass einzelne Artikel auf eigene Seiten verlinken.
